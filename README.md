## Usage

### Change Database

1. Open folder Config/Database.php : 
    - Change code : 
        ```php
            <?php
            private $dbhost = "{your localhost or ip}";
            private $dbport = "{your port}";
            private $dbusername = "{your mysql name}";
            private $dbpassword = "{your password mysql}";
            private $dbname = "{Your database name}"; 
            ...
        ```
